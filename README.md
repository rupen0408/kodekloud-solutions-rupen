# kodekloud-solutions-rupen

This is where I’m documenting my progress through the KodeKloud Engineer tasks. Instead of just doing the labs and forgetting them, I’m treating this as a daily log to track what I’ve learned and how I solved the real-world infrastructure problems presented in the xFusionCorp scenarios.

How this repo works
I'm uploading solutions day by day as I clear the tasks.

Each Day-XX folder or file contains the specific problem I was given.

I’ve included the commands I used, the logic behind them, and any "gotchas" or errors I ran into (like that annoying inventory space in Ansible!).

You'll find a mix of Linux sysadmin stuff, Ansible playbooks, and eventually some Docker/K8s as I move up the levels.
